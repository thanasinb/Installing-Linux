# Backend setup

Run the following commands

```
mkdir mqtt_mern_thanasin
cd mqtt_mern_thanasin
mkdir backend
cd backend
npm init -y
npm install express cors mongodb dotenv
sudo npm install -g nodemon
```

Open `package.json` and add `"type": "module",` below `"main"`
