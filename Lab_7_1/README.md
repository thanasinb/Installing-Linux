# Introduction to NodeJS programming for MongoDB

## Install MongoDB driver for NodeJS

Enter `mqtt_01` directory

```
cd mqtt_01
```

Install MongoDB driver for NodeJS

```
npm install mongodb
```

Create a new file `mongo_demo.js`
