## Insert a document into the collection

Modify the program using the below code and run the script

```
  var card_object = { card_id: "55 16 3B 2A", room: "39" };
  dbo.collection("rfid").insertOne(card_object, function(err, res) {
    if (err) throw err;
    console.log(res);
  });
```
